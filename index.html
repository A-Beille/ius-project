iUS Project

Voici le projet iUS. Ce projet contient des bots simples, dont l'objectif est de ne pas être tels que les bots polyvalents.


Bots présents dans ce projet

  • Arius

    • Arius est un bot permettant de parler en étant anonyme.

    Utilisation

    Pour l'utiliser, envoyez lui en message privé: l'identifiant de la personne/du salon suivi du message
    Exemple
    861290193641406464 Test enverra à Arius le message "Test". Comment trouver l'identifiant d'un salon/compte.

  • Cassius

    • Cassius est un bot... particulier.

    Utilisation

    Cassius est un bot de "modération". En réalité, si vous envoyez un message contenant un mot vulgaire, ce bot vous bannira définitivement du serveur et supprimera le salon dans lequel le message a été envoyé. Il est évident que ce bot n'st pas fait pour des utilisations dans un serveur massif et professionnel. Nous ne sommes pas responsables en cas de suppression non voulue d'un salon.

  • Sagitarius

    • Sagitarius est un bot qui répète les messages.

    Utilisation

    Sagitarius répète tous les messages. Qu'ils soient embed, avec des fichiers, sans fichiers, ou autres, il adore répéter tout et n'importe quoi. Encore une fois, ce bot peut vite devenir ennuyeux en répétant. Nous ne recommandons donc toujours pas son utilisation dans un serveur important.
